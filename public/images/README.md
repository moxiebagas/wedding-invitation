# Invitation assets

Drop the real photos/videos here. Filenames are referenced in
`src/lib/config.ts` — keep the names (or update the config) and they appear
automatically. Until then, the app shows an elegant gradient placeholder.

Expected files:

| File              | Used for                          |
| ----------------- | --------------------------------- |
| `cover.jpg`       | Cover + opening + footer hero     |
| `bride.jpg`       | "The Bride" portrait (3:4)        |
| `groom.jpg`       | "The Groom" portrait (3:4)        |
| `gallery-1.jpg` … | Wedding gallery slides            |
| `clip-1.mp4`      | A video slide in the gallery      |

Background music goes in `public/audio/backsound.mp3`.
